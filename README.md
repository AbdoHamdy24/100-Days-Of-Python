## 📝 Project Description

This repository serves as a digital log of my journey through learning Python. It contains a collection of mini-projects, automated scripts, and logic-based challenges designed to build strong programming fundamentals from the ground up. 

Each script focuses on practical application—ranging from simple text manipulation and math calculations to conditional logic and user-interactive programs.

### 📂 Current Program Directory

* **Day 1 Program:** Introduces core concepts, basic syntax, and initial code execution.
* **BMI Calculator:** A health utility script that calculates Body Mass Index based on user input for height and weight.
* **Life in Weeks:** A creative calculator that estimates your remaining time on Earth in weeks, days, and months based on a target age.
* **Tip Calculator:** A practical tool designed to split bills among friends, complete with custom tip percentages and precise rounding.
* **Leap Year Checker:** A logical program utilizing nested conditional statements to determine if any given year is a leap year.
* **Pizza Order System:** A mock e-commerce script that calculates a customer's final bill based on size preferences and optional add-ons like pepperoni or extra cheese.
